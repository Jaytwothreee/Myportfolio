XSS Penetration Test Report

Executive Summary

Cross-Site Scripting (XSS) vulnerability identified in OWASP Juice Shop. This allows malicious scripts to be executed in users' browsers.

Vulnerability Details

  • Vulnerability: Cross-Site Scripting (XSS)
  • Risk Level: Medium
  • Description: Application fails to sanitize user input, allowing injection of JavaScript code.
  • Evidence: Payload: <img src=x onerror=alert('XSS')>
  • Recommendation: Implement proper input validation and output encoding.

Steps to Reproduce

  1. Open OWASP Juice Shop
  2. Enter payload in search bar: <img src=x onerror=alert('XSS')>
  3. Submit and observe alert pop-up

Evidence / Proof of Concept

The following image shows the successful execution of the injected JavaScript payload resulting in an alert pop-up:

Impact & Recommendations

This vulnerability could allow session hijacking, phishing, or redirection to malicious sites. Apply CSP, validate inputs, and encode outputs to mitigate risks.
